## 💡 测试编写技巧

### 1. AAA 模式（Arrange-Act-Assert）

```java
@Test
void testExample() {
    // Arrange - 准备测试数据
    String input = "user_name";
    
    // Act - 执行被测试方法
    String result = DbConvertUtil.toFieldName(input);
    
    // Assert - 验证结果
    assertEquals("userName", result);
}
```

### 2. Given-When-Then 模式

```java
@Test
void testExample() {
    // Given - 给定条件
    String input = "user_name";
    
    // When - 当执行操作
    String result = DbConvertUtil.toFieldName(input);
    
    // Then - 那么应该得到
    assertEquals("userName", result);
}
```

### 3. 测试覆盖清单

对于每个方法，确保测试：
- ✅ 正常情况（Happy Path）
- ✅ 边界条件（Boundary Conditions）
- ✅ 异常情况（Exception Cases）
- ✅ 空值处理（Null Handling）
- ✅ 特殊输入（Special Inputs）
