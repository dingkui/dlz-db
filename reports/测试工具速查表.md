# 代码质量测试工具速查表

## 🚀 快速命令

| 需求 | 命令 | 耗时 |
|------|------|------|
| **运行所有检查** | `test-all.bat` | 3-5分钟 |
| **测试覆盖率** | `test-jacoco.bat` | 1-2分钟 |
| **静态分析** | `test-spotbugs.bat` | 1-2分钟 |
| **代码规范** | `test-checkstyle.bat` | 30秒 |
| **代码质量** | `test-pmd.bat` | 1分钟 |
| **查看报告** | `view-reports.bat` | 即时 |

---

## 📊 报告位置

| 工具 | 报告位置 |
|------|----------|
| **JaCoCo** | `target/reports/jacoco-aggregate/index.html` |
| **SpotBugs** | `target/reports/spotbugs/spotbugsXml.html` |
| **Checkstyle** | `target/site/checkstyle.html` |
| **PMD** | `target/site/pmd.html` |
| **CPD** | `target/site/cpd.html` |

---

## 🎯 使用场景

```
提交前     → test-jacoco.bat
PR 前      → test-all.bat
找 Bug     → test-spotbugs.bat
查规范     → test-checkstyle.bat
查质量     → test-pmd.bat
```

---

## 💡 快速打开报告

```cmd
REM Windows
start target\reports\jacoco-aggregate\index.html
start target\reports\spotbugs\spotbugsXml.html
start target\site\pmd.html

REM 或使用报告查看器
view-reports.bat
```

---

## 📝 Maven 命令

```bash
# 覆盖率
mvn test jacoco:report

# 静态分析
mvn spotbugs:check

# 代码规范
mvn checkstyle:check

# 代码质量
mvn pmd:check

# 全部
mvn clean test jacoco:report spotbugs:check checkstyle:check pmd:check
```
