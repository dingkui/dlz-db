## 🔄 迁移指南

如果你之前使用旧脚本，现在需要：

### 旧命令 → 新命令

| 旧命令 | 新命令 |
|--------|--------|
| `run-quality-checks.bat` | `cd reports && test-all.bat` |
| `quick-check.bat` | `cd reports && test-jacoco.bat && test-spotbugs.bat` |
| `view-reports.bat` | `cd reports && view-reports.bat` |

### CI/CD 配置更新

如果你的 CI/CD 使用了旧脚本，需要更新：

**旧配置**：
```yaml
script:
  - ./run-quality-checks.bat
```

**新配置**：
```yaml
script:
  - cd reports
  - ./test-all.bat
```

或者使用根目录入口：
```yaml
script:
  - ./quality-test.bat
```
