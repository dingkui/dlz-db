# 文档注意事项

> 编写 DLZ-DB 相关文档/文章时，避免以下常见错误。

## 配置类

```java
// ❌ 错误：DlzDbConfig 不存在
extends DlzDbConfig

// ✅ 正确
extends SpringDlzDbConfig
```

## likeLeft / likeRight

```java
// ❌ 错误（写反了）
likeLeft → LIKE '值%'
likeRight → LIKE '%值'

// ✅ 正确（%在左=likeLeft，%在右=likeRight）
likeLeft → LIKE '%值'    后缀匹配
likeRight → LIKE '值%'    前缀匹配
```

## 查询列

```java
// ❌ 错误：.select() 是入口工厂方法，不是条件方法
DB.pojo.selectWrapper(User.class).select(User::getId)

// ✅ 正确
DB.pojo.selectWrapper(User.class).columns(User::getId, User::getName)
```

## 标量查询

```java
// ❌ 错误：标量查询方法不接受 Class 参数
.queryStrList(String.class)
.queryStr(String.class)

// ✅ 正确
.queryStrList()
.queryStr()
```

## 不存在的 API

```java
// ❌ Params.of() — 源码中不存在
// ❌ DB.sequence() — DB 类上没有此方法
// ❌ ICacheExecutor / SpringCacheAdapter — 源码中不存在

// ✅ 正确
new JSONMap("key", value)
DBHolder.sequence("order", 100)
```

## .page() 写法

```java
// ✅ 两种都正确（ISqlPage 提供了 default 重载）
.page(1, 10)
.page(Page.build(1, 10))
```
