# 1.1 安装配置 - Spring Boot

## 环境要求

| 环境 | 版本要求 |
|------|---------|
| JDK | 8+ |
| Maven | 3.6+ |
| Spring Boot | 2.x / 3.x |

## Maven 依赖

```xml
<dependency>
    <groupId>top.dlzio</groupId>
    <artifactId>dlz-db-spring-boot-starter</artifactId>
    <version>7.0.2</version>
</dependency>
```

## yaml 配置

```yaml
# application.yml
spring:
  datasource:
    driver-class-name: com.mysql.cj.jdbc.Driver
    url: jdbc:mysql://localhost:3306/test?useSSL=false&serverTimezone=Asia/Shanghai
    username: root
    password: 123456

# DLZ-DB 配置（可选）
dlz:
  db:
     logic-delete-field: deleted
     use-db-sql: false
     helper:
       auto-update: false
       package-name: com.dlz.test.db.entity
     log:
        show-result: true
        show-run-sql: true
        show-caller: true
```

## Java 配置

```java
import com.dlz.db.spring.config.SpringDlzDbConfig;
import com.dlz.db.spring.config.SpringDlzDbProperties;
import org.springframework.boot.context.properties.EnableConfigurationProperties;
import org.springframework.context.annotation.Configuration;

@Configuration
@EnableConfigurationProperties({SpringDlzDbProperties.class})
public class DlzDbConfigs extends SpringDlzDbConfig {
}
```
