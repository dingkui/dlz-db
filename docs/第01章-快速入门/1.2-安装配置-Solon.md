# 1.2 安装配置 - Solon

## 环境要求

| 环境 | 版本要求 |
|------|---------|
| JDK | 8+ |
| Maven | 3.6+ |
| Solon | 3.x |

## Maven 依赖

```xml
<dependency>
    <groupId>top.dlzio</groupId>
    <artifactId>dlz-db-solon-plugin</artifactId>
    <version>7.0.1-2</version>
</dependency>
```

插件通过 Solon SPI 自动加载，无需手动配置 Plugin 类。

## yaml 配置

```yaml
# app.yml
dlz:
  db:
    logic-delete-field: deleted
    log:
      show-run-sql: true
      show-caller: true

# 数据源配置（Solon 原生方式）
datasource:
  default:
    jdbcUrl: jdbc:mysql://localhost:3306/test?useSSL=false
    username: root
    password: 123456
    driverClassName: com.mysql.cj.jdbc.Driver
```

完整 Solon 集成说明参见 [4.2-Solon完整集成](../第04章-框架集成/4.2-Solon完整集成.md)。
