# 1.3 五分钟上手

> 以下示例在 Spring Boot 和 Solon 下完全一致。

## 创建数据表

```sql
CREATE TABLE `user` (
  `id` bigint NOT NULL AUTO_INCREMENT,
  `name` varchar(50) DEFAULT NULL,
  `age` int DEFAULT NULL,
  `email` varchar(100) DEFAULT NULL,
  `is_deleted` tinyint DEFAULT 0,
  `create_time` datetime DEFAULT CURRENT_TIMESTAMP,
  PRIMARY KEY (`id`)
);

INSERT INTO user (name, age, email) VALUES
('张三', 25, 'zhangsan@example.com'),
('李四', 30, 'lisi@example.com'),
('王五', 28, 'wangwu@example.com');
```

## 创建实体类

```java
@Data
@TableName("user")
public class User {
    private Long id;
    private String name;
    private Integer age;
    private String email;
    private Integer isDeleted;
    private Date createTime;
}
```

## 开始使用

### 查询

```java
// 单条查询
User user = DB.Pojo.select(User.class).eq(User::getId, 1).queryOne();

// 条件查询
List<User> users = DB.Pojo.select(User.class)
    .like(name != null, User::getName, name)
    .orderByDesc(User::getCreateTime)
    .queryList();

// 分页查询
Page<User> page = DB.Pojo.select(User.class)
    .like(name != null, User::getName, name)
    .page(pageNum, pageSize)
    .queryPage();
```

### 新增

```java
User user = new User();
user.setName("张三");
DB.Pojo.insert(user);  // 自动回填主键
Long id = user.getId();
```

### 更新

```java
DB.Pojo.update(user).eq(User::getId, user.getId()).execute();
```

### 删除

```java
DB.Pojo.delete(User.class).eq(User::getId, id).execute();
```
