# AI 速读 · Solon 版

> Java 持久层框架。静态入口 `DB.`，链式 API，无 Mapper / XML 样板。
> 本文档为 AI 生成代码的唯一规范来源。

## 心智模型

```
DB.Pojo     —— 有 Bean 的 CRUD（首选）
DB.Table    —— 动态表名（无 Bean 或分表）
DB.Jdbc     —— 一行原生 SQL，占位符 ?
DB.Sql      —— 预设/复杂 SQL，占位符 #{}
DB.Batch    —— 批量写入
DB.Dynamic  —— 数据源切换作用域
```

## 入口职责

| 入口 | 场景 | 占位符 |
|---|---|---|
| `DB.Pojo.select(T.class)` | 简单 CRUD | 条件构造器 |
| `DB.Table.select("t_202401")` | 运行时表名查询 | 条件构造器 |
| `DB.Jdbc.select("SELECT ... WHERE x=?", v)` | 一次性 SQL | `?` |
| `DB.Sql.select("key.xxx")` | 预设 SQL / 复杂 SQL | `#{key}` |

## 条件方法

| 方法 | SQL |
|---|---|
| `eq / ne / gt / ge / lt / le` | `= / <> / > / >= / < / <=` |
| `isn / isnn` | `IS NULL / IS NOT NULL` |
| `in / ni` | `IN / NOT IN` |
| `bt / nb` | `BETWEEN / NOT BETWEEN` |
| `lk / ll / lr / nl` | `LIKE` 四种变体（**不是** `like`） |
| `columns(...)` | 指定 SELECT 字段（**不是** `select(...)`） |
| `or(o -> ...)` / `and(a -> ...)` | 嵌套括号 |
| `sql("SQL #{k}", JSONMap)` | 自定义 SQL |

## 返回值规则

| 方法 | 返回 |
|---|---|
| `queryBean()` / `queryBeanList()` / `queryBeanPage()` | `T` / `List<T>` / `Page<T>` |
| `queryOne()` / `queryList()` / `queryPage()` | `ResultMap` / `List<ResultMap>` / `Page<ResultMap>` |
| `queryOne(C)` / `queryList(C)` / `queryPage(C)` | 指定类型 |
| `count()` | `int` |

**规则**：带 `Bean` → Bean；不带 → `ResultMap`；带 `(Class)` → 指定类型。

## 硬约束

1. **无 Mapper / DAO**：不写 `@Mapper`、不 `@Autowired UserDao`。CRUD 直接 `DB.Pojo.*`。
2. **无 Wrapper 类**：不写 `new LambdaQueryWrapper<>()`。用链式 `DB.Pojo.select(...)`。
3. **无 `selectList` 等 MP 方法**：它们不存在。`selectById` / `deleteById` 等直接操作方法可用。
4. **占位符不可混用**：`DB.Jdbc` 用 `?`；`DB.Sql` / `sql(...)` 用 `#{key}`。
5. **`queryList()` 返回 `List<ResultMap>`**，要 Bean 用 `queryBeanList()`。
6. **写操作必须 `.execute()` 结尾**。
7. **预设 SQL key 必须以 `"key."` 开头**。
8. **`in(field, value)` 的 value** 必须是 `List` / CSV 字符串 / 子查询，**不可传单值**。
9. **`queryOne()` 返回 `ResultMap`**，要 Bean 用 `queryBean()`（对 `queryList / queryPage` 同理）。
10. **LIKE 操作是 `lk()` 不是 `like()`**：`lk`（%v%）、`ll`（v%）、`lr`（%v）、`nl`（NOT LIKE）。
11. **指定查询字段用 `columns()` 不是 `select()`**：`.columns(User::getId, User::getName)`。
12. **`count()` 返回 `int` 不是 `long`**。

## Entity 约定

- 表名：默认驼峰转下划线。自定义：`@TableName("t_user")`
- 逻辑删除：Bean 含 `isDeleted` 字段时自动注入 `IS_DELETED=0`

## 事务

Solon 下使用 `@Tran` 注解而非 `@Transactional`。
